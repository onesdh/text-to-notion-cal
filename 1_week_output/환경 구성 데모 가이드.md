# 환경 구성 데모 가이드

## 1. 가상환경 생성 (Conda)

Conda를 사용하여 새로운 가상환경을 생성하고 활성화합니다.

```bash
# Conda 환경 생성 (python 3.12)
conda create -n text_to_cal python=3.12 -y

# 환경 활성화
conda activate text_to_cal
```

## 2. 필수 패키지 설치

`requirements.txt` 파일에 명시된 라이브러리들을 설치합니다.

```bash
pip install -r requirements.txt
```

## 3. 환경 변수 설정

- 프로젝트 루트 디렉토리에 `.env` 파일 생성
- `.env` 파일에 다음과 같이 API 키 추가
```env
OLLAMA_URL = "YOUR_OLLAMA_SERVER_URL"
OLLAMA_LLM_MODEL = "YOUR_OLLAMA_LLM_MODEL"
OPENAI_API_KEY = "YOUR_OPENAI_API_KEY"
NOTION_TOKEN = "YOUR_NOTION_INTEGRATION_TOKEN"
DATABASE_ID = "YOUR_NOTION_DATABASE_ID"
```
