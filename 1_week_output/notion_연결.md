1. Notion API 초기 설정
1-1. Notion Integration 생성

Notion Developers
https://www.notion.so/profile/integrations
 접속

New Integration 클릭 → 이름 지정

권한 설정:

Read, Write 권한 체크 (데이터베이스 생성/수정 가능)

Integration Token 발급 → 안전하게 보관

1-2. 데이터베이스 생성

Notion에서 새 페이지 생성

Database → Table 선택

컬럼 구성:

Title (제목) → 이벤트 이름

Date (날짜) → 날짜/시간 포함

Description (설명) → 일정 내용

필요 시 Tags 또는 Status 추가

1-3. Integration과 데이터베이스 연결

데이터베이스 우측 상단 Share → Invite 클릭

Integration 선택 → 권한 부여
