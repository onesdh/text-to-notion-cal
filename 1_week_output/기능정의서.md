# 프로젝트 이름
text_to_calendar

# 프로젝트 목표
- 사용자가 자연어로 작성한 일정 정보를 자동으로 캘린더에 추가하는 기능 구현
- LangGraph를 활용하여 text -> calendar 변환 파이프라인 구현
- 단기간 토이 프로젝트로 핵심 기능 구현에 우선 집중

# 주요 기능 정의
## 1. 입력
- 사용자로부터 일정 정보를 자연어 텍스트로 입력 받음
    - 예시 : 
        - "내일 오전 10시에 회의"
        - "다음 주 금요일 2시에 치과 예약"

## 2. 일정 정보 추출
- LangGraph를 활용하여 텍스트에서 날짜, 시간, 이벤트 내용을 추출
    - 추출 데이터 예시 :
        - 제목: "회의"
        - 시작 시간: "2025-10-14T10:00:00+09:00"
        - 종료 시간: "2025-10-14T11:00:00+09:00"
        - 내용 : "팀 미팅"

## 3. 캘린더 생성/추가 함수
- 추출된 데이터를 바탕으로 캘린더 이벤트 객체 생성
- Calendar API 를 활용하여 캘린더에 일정 추가
    - Google Calendar API, Apple Calendar API 등 선택 가능

## 4. 기능요약
| 기능        | 입력       | 처리                   | 출력               |
| --------- | -------- | -------------------- | ---------------- |
| 자연어 일정 입력 | 텍스트      | -                    | 텍스트 그대로          |
| 일정 정보 추출  | 텍스트      | 날짜, 시간, 제목 파싱        | JSON 이벤트         |
| 캘린더 추가    | JSON 이벤트 | 캘린더 객체 생성, 파일/API 추가 | 캘린더 이벤트 추가 완료 알림 |


## Add. 노션 캘린더 선택이유

- 전반적인 캘린더 기능은 구글이 잘되어 있으나 구글의 경우 oauth 인증이 필요하며 
개인 프로젝트로 사용하기에는 복잡한 점이 많음
- 노션 캘린더는 노션 데이터베이스의 한 형태로 존재하며, 노션 API를 통해 손쉽게 접근 가능하며 
간단한 토큰 기반 인증으로 개인 프로젝트에 적합함
